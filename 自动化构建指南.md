# 🎉 截历 - 完全自动化 APK 构建方案

## 🚀 方案一：GitHub 网页上传（最简单！

### 🔥 一键开始（3 步完成！

#### 第 1 步：创建仓库
👉 点击：https://github.com/new

填写：
- Repository name: `jieli-screenshot-manager`
- 选择 Public 或 Private
- **不要**勾选任何初始化选项
- 点击：**Create repository**

#### 第 2 步：上传文件
1. 在新页面点击：**uploading an existing file**
2. 打开文件夹：`C:\Users\Administrator\.qclaw\workspace\screenshot-manager`
3. **全选所有文件**（Ctrl+A）
4. **拖拽**到 GitHub 上传区域
5. 等待上传完成
6. 填写提交信息：`Initial commit`
7. 点击：**Commit changes**

#### 第 3 步：下载 APK！
1. 点击仓库顶部 **Actions** 标签
2. 等待 3-5 分钟，构建自动完成！
3. 在构建详情页下载 **截历-APK**！

## 🌟 方案二：GitHub Desktop（带 GUI 的 Git）

### 下载 GitHub Desktop
👉 https://desktop.github.com/

### 使用步骤
1. 安装后打开 GitHub Desktop
2. File -> Add Local Repository
3. 选择文件夹：`C:\Users\Administrator\.qclaw\workspace\screenshot-manager`
4. 点击 "Create repository"
5. 发布到 GitHub
6. GitHub Actions 自动构建！
