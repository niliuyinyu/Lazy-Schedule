# 快速上传指南

## 🚀 最快方式（无需安装任何东西！

### 第 1 步：创建 GitHub 仓库
- 访问：https://github.com/new
- 仓库名称：jieli-screenshot-manager
- 点击：Create repository

### 第 2 步：上传文件
- 点击：uploading an existing file
- 打开文件夹：C:\Users\Administrator\.qclaw\workspace\screenshot-manager
- 全选所有文件（Ctrl+A）
- 拖拽到 GitHub
- 提交！

### 第 3 步：下载 APK
- 等待 3-5 分钟
- 进入仓库 Actions 标签
- 下载 APK！
